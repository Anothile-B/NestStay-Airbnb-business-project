# NestStay – Airbnb Web Application

---

# Project Overview
NestStay is a front-end web application inspired by Airbnb, built to provide users with a seamless experience when browsing accommodations. 
The app focuses on clean design, usability, and responsive layout for both desktop and mobile users.

# Features
- Browse available properties
- View property details such images, pricing, descriptions
- Booking system with date selection
- Map integration for property locations
- Payment integration
- Reviews system
- Clean and user-friendly interface
- Fully responsive design
- Easy navigation between pages
- Modern UI inspired by Airbnb

---

# Technologies Used
- HTML5: Structure of the application
- CSS3: Styling and layout
- JavaScript: Interactivity and dynamic content
